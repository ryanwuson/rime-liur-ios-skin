# 蝦米輸入法皮膚設計器使用指南

歡迎使用 **RIME 蝦米輸入法皮膚設計器**！這是一個專為 iOS 元書輸入法 + 蝦米方案用戶打造的視覺化皮膚編輯工具，您無需修改複雜的代碼，即可親手設計出最符合個人習慣的鍵盤皮膚。

蝦米輸入方案：https://ryanwuson.github.io/rime-liur/#/

蝦米輸入法皮膚設計器：https://ryanwuson.github.io/rime-liur-ios-skin/

---

## 🚀 快速上手步驟

1. **調整操作習慣**：在「按鍵佈局」中設定適合您的空白鍵長度與滑動功能。
2. **配置常用工具**：在「工具配置」中將常用的編輯、符號等功能放在最順手的位置。
3. **挑選基礎樣式**：從「基礎外觀」，選定您喜歡的配色方案。
4. **檢查深色模式**：記得點擊左上角的太陽/月亮圖標，設定深色模式下的視覺效果。
5. **導出並安裝**：
   - 點擊 **「導出配置」** 下載 `.cskin` 檔案。
   - 並透過 iOS 元書輸入法的長按鍵盤皮膚並 **「運行 main.jasonet」** 即可使用。

---
*💡 提示：若不小心把介面調亂了，每個區域都有「還原預設值」按鈕可以隨時重來。*

## 🖥️ 介面功能導覽

各項功能說明如下：

### 一、預覽切換區

您可以切換不同的鍵盤狀態（如 26 鍵、數字、符號、Emoji）、英文大小寫、候選字，也可點按及長按鍵盤顯示按鍵氣泡；透過切換 **淺色/深色模式** ，確保皮膚在各種情境下都能完美呈現；亦有 **重置按鈕**，讓您一鍵還原預設裝態。

### 二、即時預覽面板

即時反映您的所有設定。您可以直接看到鍵盤顏色、字體大小以及工具列的配置結果。

### 三、按鍵佈局

在此分頁可以自定義鍵盤的佈局樣式，例如：

- 空白鍵樣式（短/長）。
- 單手模式設定（左手/右手持機）。
- 全域或分行的滑動與長按功能開關。

### 四、工具配置

您可以自定義鍵盤上方的 **工具列（最多10個）：

- 自由排序各項工具（如剪下、複製、貼上、切換繁簡等）。
- 選擇要顯示在工具列上的功能按鍵。

### 五、基礎外觀

最核心的視覺設定區：

- 設定鍵盤背景色、按鍵背景色、文字顏色。
- 調整顏色的 **透明度 (Opacity)**。
- 變更文字字體大小。

### 六、進階微調

針對細節控設計：針對各個鍵盤，提供更細緻的參數調整。

### 七、編輯皮膚資訊

點擊後可設定皮膚的名稱、作者資訊以及說明，這些資訊將包含在導出的皮膚包中。

### 八、匯入配置 (.cskin)

如果您已有現成的 `.cskin` 檔案或進度，可以點擊此按鈕將其再次匯入設計器進行修改。

### 九、導出配置 (.cskin)

**設計完成！** 點擊此按鈕即可導出專屬的皮膚配置文件。
